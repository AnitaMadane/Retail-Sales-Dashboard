# Retail Sales Dashboard (Power BI)

## 📊 Project Overview
This project analyzes retail sales data using **Power BI**.  
The dashboard provides insights into total sales, sales trends over time, and breakdowns by region and product.

## 🗂 Dataset
- Source: `retails.xlsx`
- Fields: Customer, OrderDate, OrderID, Product, Quantity, Region, Sales, UnitPrice

## 📈 Dashboard Features
1. **Total Sales Card** – Displays overall sales value.
2. **Sales Trend Line Chart** – Shows sales performance over time.
3. **Sales by Region Chart** – Compares sales across North, South, East, West.
4. **Sales by Product Chart** – Compares Laptop, Mobile, Tablet sales.
5. **Interactive Slicers** – Filters by Region and Product.
